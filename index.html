What is this project about?

This is the first big project I made while studying in the Polytechic University of Puerto Rico. I decided to bring this intermediate project to a more advanced setting by recreating it with AWS.
For this project I used various services from AWS:

+ Cognito for sign up and sign in functionalities.
+ Dynamo DB to store and recover the employee's information.
+ Lambda to incorporate python for the information entries.
+ API Gateway for API management and unification of Front and Back ends.
+ CodeCommit to bring in the repository which is what you see here.
